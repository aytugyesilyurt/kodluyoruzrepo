# ÖDEV 1 ( Insertion Sort )

> 1 ) [22,27,16,2,18,6] verilen dizinin sort türüne göre aşamalarını yazınız.
>
> - [2,27,16,22,18,6]
> - [2,6,16,22,18,27]
> - [2,6,16,18,22,27]

> 2 ) Big-O gösterimini yazınız.
>
> - O(n^2)

> 3 ) Time Complexity
>
> - Worst case : O(n^2)
> - Average case : O(n^2)
> - Best case : O(n)

> 4 ) Dizi sıralandıktan sonra 18 sayısı hangi case kapsamına girer?
>
> - Ortanca değer olduğu için Average Case.

# ÖDEV 2 ( Merge Sort)

> 1 ) [16,21,11,8,12,22] dizinin sort türüne göre aşamalarını yazınız.
>
> - [16,21,11] ... [8,12,22]
> - [16] ... [21,11] ... [8,12] ... [22]
> - [16] ... [21] ... [11] ... [8] ... [12] ... [22]
> - [16] ... [11,21] ... [8,12] ... [22]
> - [11,16,21] ... [8,12,22]
> - [8,11,12,16,21,22]

> 2 ) Big-O gösteriminiz yazınız.
>
> - O(nlogn)
